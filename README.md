# Academic Notes Master Elite

> Built by **Rudolph Chisomo Ussi (Russ)** — MUST Year 1, Computer Systems & Security

A specialised AI study skill for Claude that generates exam-ready `.docx` study 
packages for MUST Year 1 Semester 2 students.

**Supported Modules:**
MATH-122 · MATH-123 · PHYS-122 · COMP-122 · INNV-122 · TBUC

---

## 🆕 Fresh Install (New Users)

If you have never used this skill before:

1. Download `academic notes master elite.md` from this repo
2. Open **Claude** and go to your **Skills** section
3. Upload the downloaded file
4. Done! Say `/academic-notes-master-elite` in Claude to activate it ✅

---

## 🔄 Updating to the Latest Version

If you already have the skill installed:

1. Download the latest `academic notes master elite.md` from this repo
2. Go to your **Skills** section in Claude
3. **Delete your old version** of the skill
4. Upload the newly downloaded file
5. You're now on the latest version ✅

---

## 🩹 Patching from a Pre-GitHub Version

If you received the skill file before it was on GitHub:

1. Download the latest `academic notes master elite.md` from this repo
2. Go to your **Skills** section in Claude
3. **Delete the old skill file** you received previously
4. Upload the newly downloaded file
5. You are now on the official version ✅
